#  Student Performance Analysis

## 📌 Project Overview
This project presents a comprehensive statistical analysis of student academic performance using the *Students Performance Dataset*.  
The objective is to understand how **demographic, socioeconomic, and educational factors** influence students’ scores in **math, reading, and writing**.

The analysis applies **exploratory data analysis (EDA)** and **statistical hypothesis testing** to uncover patterns and significant relationships within the data.

---

##  Objectives
The goals of this project are to:
- Explore student performance across different attributes
- Identify patterns and disparities in academic outcomes
- Test for statistically significant differences between groups
- Draw data-driven conclusions using sound statistical methods
- Provide a foundation for building a student performance analysis system


##  Analysis Summary

### 1️⃣ Exploratory Data Analysis (EDA)
- Computation of summary statistics (mean, median, standard deviation)
- Distribution analysis for all three subjects
- Group-wise comparisons across categorical attributes
- Visualizations to identify trends and variability

---

### 2️⃣ Gender vs Student Performance
- Comparison of male and female students across subjects
- Small but observable differences in average scores
- Performance varies slightly by subject

---

### 3️⃣ Race/Ethnicity vs Student Performance
- Differences in average scores observed across racial/ethnic groups
- Performance gaps suggest the influence of broader social and educational factors

---

### 4️⃣ Parental Level of Education vs Student Performance
- Students whose parents have higher education levels tend to perform better
- Clear upward trend in mean scores with increasing parental education

---

### 5️⃣ Lunch Type vs Student Performance
- Students receiving **standard lunch** consistently outperform those on **free/reduced lunch**
- One-way **ANOVA** tests confirm statistically significant differences across all subjects

---

### 6️⃣ Test Preparation Course vs Student Performance
- Students who completed the test preparation course score higher on average
- Indicates a positive association between preparation and academic performance

---

## 📈 Statistical Methods Used
- Descriptive statistics
- Group-wise mean comparisons
- One-way ANOVA
- Data visualization for comparative analysis

---

## 🧠 Key Insights
- Socioeconomic factors (lunch type, parental education) strongly relate to performance
- Academic preparation has a measurable positive effect
- Performance disparities exist across demographic groups
- Student performance is influenced by multiple interacting factors rather than a single variable

##  Conclusion
This project provides a comprehensive statistical examination of factors affecting student performance.  
The findings highlight the importance of socioeconomic background, parental education, and academic preparation in shaping educational outcomes.
